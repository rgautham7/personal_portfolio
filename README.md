# Personal Portfolio

This is a personal portfolio website built with HTML, CSS, and JavaScript. It showcases the developer's skills, projects, and contact information.

## Features

- Responsive design
- About section with skills and education
- Projects showcase
- Contact information
- Tech stack display
- Dark theme

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Vite (for build and development)

## Setup and Installation

1. Clone the repository:
   ```
   git clone https://github.com/rgautham7/personal_portfolio.git
   ```

2. Navigate to the project directory:
   ```
   cd personal_portfolio
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Build for production:
   ```
   npm run build
   ```

## Project Structure

- `index.html`: The main HTML file containing the structure of the portfolio.
- `src/main.js`: The main JavaScript file that imports all the necessary CSS files.
- `styles/`: Contains all the CSS files for styling the portfolio.
  - `components/`: CSS files for individual components like header, hero, about, etc.
- `public/`: Contains public assets like images.

## Customization

To customize the portfolio for your own use:

1. Update the personal information in `index.html`.
2. Replace project images and descriptions in the "Projects" section.
3. Update the contact information in the "Get in Touch" section.
4. Modify the styles in the CSS files to match your preferred color scheme and layout.
